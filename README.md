# Astra Test Task

Решение двух тестовых задач на Python без внешних зависимостей.

## Файлы

- `task1_grades.py` - функция `process_grades(records: list[str]) -> dict`
- `task2_streak.py` - функция `longest_increasing_streak(nums: list[int]) -> dict`

## Запуск

Требуется Python 3.9+.

Запустить проверочные примеры для первой задачи:

```bash
python task1_grades.py
```

Запустить проверочные примеры для второй задачи:

```bash
python task2_streak.py
```

Внутри файлов есть несколько `assert`-проверок. Если проверка не пройдет, Python завершит выполнение с ошибкой.

## Примечание

В примере первой задачи из условия среднее для валидных оценок `85, 42, 90, 70` равно `71.8`, поэтому в проверке используется это значение.
